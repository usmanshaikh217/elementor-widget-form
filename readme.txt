Requires at least: 4.6

== Description ==

This is custom contact form with elementor, created custom post type with taxonony and submit all entry in 'Submission' Post Type.


== Steps ==

1) Download Plugin from GitHub
2) Upload Plugin
3) Go to WP Dashboard, Check left bar 'Submission', Add Submission Categories (career form, contact form, etc).
4) Create New Page in Elementor and Edit mode as Elementor, search 'Custom Widget Form' and drop in Page Content.
5) Page Publish
6) Check Public Page, form willbe display.
7) Submit test entry
8) check entry in Dashboard > Submission
